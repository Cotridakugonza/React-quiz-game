# Josiah Game 🎮

A fun educational game for kids featuring:

## Features
- 🎮 Bible Trivia Q&A
- 🌍 Fun Facts Quiz (safe for kids)
- 🧠 Levels (Easy, Medium, Expert)
- 🏆 Score system
- 🔊 Sound effects
- 👤 Player dashboard (Josiah profile)

## Tech Stack
- React (Vite)
- Node.js + Express
- MongoDB
